# AI Agent Guidance Index

This repository uses a dedicated `agent-guides/` directory to store AI guidance documents that help agents understand the project structure, stack, and conventions.

## Guide files

- `agent-guides/README.md` — overview of the agent guides and how to use them.
- `agent-guides/project-overview.md` — project architecture, key folders, and feature locations.
- `agent-guides/tech-stack.md` — technology stack, libraries, and configuration details.
- `agent-guides/agent-guidance.md` — workflow and best practices for AI agents editing this repo.

## How to use

When you are asked to work on this repository:

1. Open `AGENTS.md` first.
2. Read `agent-guides/README.md`.
3. Review `agent-guides/project-overview.md` and `agent-guides/tech-stack.md`.
4. Follow the instructions in `agent-guides/agent-guidance.md`.

## Purpose

This file is the entrypoint for AI agents and human collaborators who want a quick orientation on the repository’s documentation and agent workflow.
