# Agent Guides

This directory contains guidance files for AI agents working on the AutoMobileShop repository.

Files:

- `project-overview.md` — high-level repo structure and key folders.
- `tech-stack.md` — tech stack, config, and library patterns.
- `agent-guidance.md` — instructions for AI agents on how to work with this repo.

**Usage:**

When asked to edit or analyze this project, read these files first. They summarize the architecture, conventions, and the most relevant locations for common changes.
